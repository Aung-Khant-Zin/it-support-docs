# New User IT Onboarding Checklist

Use this checklist when setting up IT for a new employee joining the organisation.

---

## Pre-Arrival (Before Day 1)

- [ ] Create user account in Active Directory
- [ ] Assign user to appropriate security groups and OUs
- [ ] Create Microsoft 365 account and assign licence
- [ ] Set up company email address
- [ ] Prepare and image laptop/desktop with standard OS build
- [ ] Install required software (Microsoft 365, VPN client, antivirus, department-specific apps)
- [ ] Apply latest Windows Updates and security patches
- [ ] Configure email client (Outlook) with company account
- [ ] Set up VPN access if required (remote/hybrid role)
- [ ] Prepare peripheral devices (mouse, keyboard, monitor, headset if needed)
- [ ] Label device with asset tag and register in IT asset management system

---

## Day 1 — Device Handover

- [ ] Hand over device to new user
- [ ] Confirm user can log in with temporary password
- [ ] Prompt user to change password on first login
- [ ] Walk user through MFA (Multi-Factor Authentication) setup
- [ ] Confirm email is working — send test email
- [ ] Confirm Microsoft Teams is accessible and signed in
- [ ] Confirm OneDrive is syncing correctly
- [ ] Connect to office Wi-Fi or LAN and confirm internet access
- [ ] Test VPN connection if applicable
- [ ] Show user how to connect to network printers
- [ ] Explain how to raise IT support tickets (helpdesk portal / email / phone)

---

## Software Setup

- [ ] Microsoft 365 (Outlook, Teams, OneDrive, SharePoint)
- [ ] VPN client (configured and tested)
- [ ] Antivirus / endpoint protection (confirmed active)
- [ ] Department-specific software (as per manager's request)
- [ ] Browser (Edge/Chrome) configured with bookmarks if required
- [ ] Remote desktop tools if applicable

---

## Security & Compliance

- [ ] User briefed on password policy (complexity, expiry)
- [ ] MFA enabled on Microsoft 365 account
- [ ] User signed acceptable use policy (if required)
- [ ] Laptop encryption (BitLocker) confirmed active
- [ ] Antivirus confirmed active and up to date
- [ ] User advised not to install unauthorised software

---

## Post-Onboarding (End of Week 1)

- [ ] Follow up with user — any IT issues or questions?
- [ ] Confirm all software is working as expected
- [ ] Update asset register with device assigned to user
- [ ] Close onboarding ticket in helpdesk system
